Welcome to Fast5's documentation!
=================================

This package comprises an API to HDF containers used by the research groups
within Oxford Nanopore Technologies. It complements the
`official API <https://github.com/nanoporetech/ont_fast5_api>`_. Reading and
writing of read files can be accomplished as well as reading of bulk .fast5
files.


Installation
------------

The package can been installed via pip with:

.. code-block:: bash

    pip install fast5_research

See _`examples` for details of basic use.
